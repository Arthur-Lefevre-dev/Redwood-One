    Modifier le film
    
        
    
        
        
        
        
    
          Si vous éditez un épisode de série : les champs série
          ci‑dessous doivent être strictement identiques sur
          chaque épisode de cette série dans le catalogue.

    
          Titre affiché de la série — le nom de la série telle
          qu’elle apparaît pour les spectateurs (ex. le titre de la série
          entière), pas le titre de cet épisode seul.

    
          Clé série — un identifiant technique unique pour
          toute la série, le même sur tous les épisodes (ex.
          tv-57243, doctor-who).

    
          Saison et épisode changent à chaque
          épisode ; une fois renseignés, cliquez Enregistrer.
        
    
        
    
          
          
          
          
          
          
          
          
        
    
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
        
    
          Format tt + chiffres. Utile pour l’enrichissement IMDb
          API et les épisodes.
        
    
        
        
        
    
          S’affichent avant les vidéos récupérées via TMDB.
          Laissez vide pour n’utiliser que TMDB. Pour
          One Battle After Another, renseignez aussi l’ID TMDB ci-dessus.
        
    
        
        
        
        
        
    
          
          
        
    
      
    
    
    

    
    
      
    
        
    
          
    Fiche série et saisons
    
          
        
    
        
    
          
    
          
    
            Même clé série que sur les épisodes. En haut :
            affiche globale de la page série et texte sous le
            titre ; plus bas : une
            affiche par saison (indépendante) et note
            optionnelle.
          
    
          
          
          
          
          
    
            Aucune clé série n’est encore enregistrée sur les
            épisodes de ce bloc. Vous pouvez quand même définir une affiche :
            saisissez la clé (la même que vous mettrez dans
            Modifier sur chaque épisode), puis enregistrez. Ce
            bandeau disparaît dès que le champ ci‑dessus est rempli.
          
    
          
    
            Page série (grand poster + texte sous le titre)
          
    
          
          
          
          
          
          
    
            Utilisez bien Enregistrer la fiche globale pour
            cette section — le bouton du bas ne concerne que les saisons.
            Laissez les deux champs vides et enregistrez pour supprimer la fiche
            globale.
